Portfolio personal construido con Astro y Tailwind CSS, mostrando experiencia
profesional, proyectos y formas de contacto.

## 🚀 Stack

- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)
- TypeScript

## 🧑‍💻 Desarrollo local

```bash
pnpm install
pnpm dev
```

## 📦 Build

```bash
pnpm build
```
